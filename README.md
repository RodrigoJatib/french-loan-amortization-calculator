# French Loan Amortization Calculator

This project implements a loan amortization schedule using the French system. 
It calculates fixed installments, interest, and principal for each period.

## Features
- Fixed installment calculation
- Amortization table generation
- Total interest and principal summary

---

## Calculadora de Amortización – Sistema Francés

Este proyecto implementa una calculadora de cuadro de amortización utilizando el sistema francés. 
Calcula la cuota fija, el interés y la amortización de cada período.

### Funcionalidades
- Cálculo de cuota fija
- Generación de cuadro de amortización
- Resumen de interés total y capital
